28  /*
29  * Organization of workspace in Lapack.
30  * We allow one of the following arguments in a number of Lapack functions
31  * - minimal_workspace() : the function allocates the minimum workspace required for the function
32  * - optimal_workspace() : the function allocates the amount of workspace that allows optimal
33  * execution.
34  * - workspace( work ) : the function uses the workspace array in work.
35  * - workspace( rwork, work ) : the function uses a real array rwork and a compolex array work as
36  * workspace. (There are Lapack functions for complex matrices
37  * that require two workarrays)
38  * */
