133 /*
134  * *********************************************************************
135  * The superlumt_options_t structure contains the shared variables used
136  * for factorization, which are passed to each thread.
137  * *********************************************************************
138  *
139  * nprocs (int)
140  * Number of processes (or threads) to be spawned and used to perform
141  * the LU factorization by pdgstrf().
142  *
143  * fact (fact_t)
144  * Specifies whether or not the factored form of the matrix
145  * A is supplied on entry, and if not, whether the matrix A should
146  * be equilibrated before it is factored.
147  * = FACTORED: On entry, L, U, perm_r and perm_c contain the
148  * factored form of A. If equed is not 'N', the matrix A has
149  * been equilibrated with scaling factors R and C.
150  * A, L, U, perm_r are not modified.
151  * = DOFACT: The matrix A will be factored, and the factors will be
152  * stored in L and U.
153  * = EQUILIBRATE: The matrix A will be equilibrated if necessary, then
154  * factored into L and U.
155  *
156  * trans (trans_t)
157  * Specifies the form of the system of equations:
158  * = NOTRANS: A * X = B (No transpose)
159  * = TRANS: A**T * X = B (Transpose)
160  * = CONJ: A**H * X = B (Transpose)
161  *
162  * refact (yes_no_t)
163  * Specifies whether this is first time or subsequent factorization.
164  * = NO: this factorization is treated as the first one;
165  * = YES: it means that a factorization was performed prior to this
166  * one. Therefore, this factorization will re-use some
167  * existing data structures, such as L and U storage, column
168  * elimination tree, and the symbolic information of the
169  * Householder matrix.
170  *
171  * panel_size (int)
172  * A panel consists of at most panel_size consecutive columns.
173  *
174  * relax (int)
175  * To control degree of relaxing supernodes. If the number
176  * of nodes (columns) in a subtree of the elimination tree is less
177  * than relax, this subtree is considered as one supernode,
178  * regardless of the row structures of those columns.
179  *
180  * diag_pivot_thresh (double)
181  * Diagonal pivoting threshold. At step j of the Gaussian elimination,
182  * if abs(A_jj) >= diag_pivot_thresh * (max_(i>=j) abs(A_ij)),
183  * use A_jj as pivot, else use A_ij with maximum magnitude.
184  * 0 <= diag_pivot_thresh <= 1. The default value is 1,
185  * corresponding to partial pivoting.
186  *
187  * drop_tol (double) (NOT IMPLEMENTED)
188  * Drop tolerance parameter. At step j of the Gaussian elimination,
189  * if abs(A_ij)/(max_i abs(A_ij)) < drop_tol, drop entry A_ij.
190  * 0 <= drop_tol <= 1. The default value of drop_tol is 0,
191  * corresponding to not dropping any entry.
192  *
193  * usepr (yes_no_t)
194  * Whether the pivoting will use perm_r specified by the user.
195  * = YES: use perm_r; perm_r is input, unchanged on exit.
196  * = NO: perm_r is determined by partial pivoting, and is output.
197  *
198  * SymmetricMode (yest_no_t)
199  * Specifies whether to use symmetric mode.
200  *
201  * PrintStat (yes_no_t)
202  * Specifies whether to print solver's statistics.
203  *
204  * perm_c (int*) dimension A->ncol
205  * Column permutation vector, which defines the
206  * permutation matrix Pc; perm_c[i] = j means column i of A is
207  * in position j in A*Pc.
208  * When search for diagonal, perm_c[*] is applied to the
209  * row subscripts of A, so that diagonal threshold pivoting
210  * can find the diagonal of A, instead of that of A*Pc.
211  *
212  * perm_r (int*) dimension A->nrow
213  * Row permutation vector which defines the permutation matrix Pr,
214  * perm_r[i] = j means row i of A is in position j in Pr*A.
215  * If usepr = NO, perm_r is output argument;
216  * If usepr = YES, the pivoting routine will try to use the input
217  * perm_r, unless a certain threshold criterion is violated.
218  * In that case, perm_r is overwritten by a new permutation
219  * determined by partial pivoting or diagonal threshold pivoting.
220  *
221  * work (void*) of size lwork
222  * User-supplied work space and space for the output data structures.
223  * Not referenced if lwork = 0;
224  *
225  * lwork (int)
226  * Specifies the length of work array.
227  * = 0: allocate space internally by system malloc;
228  * > 0: use user-supplied work array of length lwork in bytes,
229  * returns error if space runs out.
230  * = -1: the routine guesses the amount of space needed without
231  * performing the factorization, and returns it in
232  * superlu_memusage->total_needed; no other side effects.
233  *
234  * etree (int*)
235  * Elimination tree of A'*A, dimension A->ncol.
236  * Note: etree is a vector of parent pointers for a forest whose
237  * vertices are the integers 0 to A->ncol-1; etree[root]==A->ncol.
238  * On input, the columns of A should be permutated so that the
239  * etree is in a certain postorder.
240  *
241  * colcnt_h (int*)
242  * Column colunts of the Householder matrix.
243  *
244  * part_super_h (int*)
245  * Partition of the supernodes in the Householder matrix.
246  * part_super_h[k] = size of the supernode beginning at column k;
247  * = 0, elsewhere.
248  *
249  *
250  */
