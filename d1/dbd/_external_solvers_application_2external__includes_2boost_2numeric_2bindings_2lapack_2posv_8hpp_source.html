42  /*
43  * posv() computes the solution to a system of linear equations
44  * A * X = B, where A is an N-by-N symmetric or Hermitian positive
45  * definite matrix and X and B are N-by-NRHS matrices.
46  *
47  * The Cholesky decomposition is used to factor A as
48  * A = U^T * U or A = U^H * U, if UPLO = 'U',
49  * A = L * L^T or A = L * L^H, if UPLO = 'L',
50  * where U is an upper triangular matrix and L is a lower triangular
51  * matrix. The factored form of A is then used to solve the system of
52  * equations A * X = B.
53  *
54  * If UPLO = 'U', the leading N-by-N upper triangular part of A
55  * contains the upper triangular part of the matrix A, and the
56  * strictly lower triangular part of A is not referenced.
57  * If UPLO = 'L', the leading N-by-N lower triangular part of A
58  * contains the lower triangular part of the matrix A, and the
59  * strictly upper triangular part of A is not referenced.
60  */
