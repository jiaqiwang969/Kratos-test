Detailed Description

template<class TDataType>
class Kratos::Variable< TDataType >

Variable class contains all information needed to store and retrive data from a data container.

Variable class contains all information needed to store and retrive data from a data container. It contains key value which is needed for searching in data container. Also a zero value to use as a default vlaue in container. Finally it has the type of the Variable as its template parameter so container can find it int its relative part.
