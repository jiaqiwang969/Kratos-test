41  /*
42  * gesv() computes the solution to a system of linear equations
43  * A * X = B, where A is an N-by-N matrix and X and B are N-by-NRHS
44  * matrices.
45  *
46  * The LU decomposition with partial pivoting and row interchanges
47  * is used to factor A as A = P * L * U, where P is a permutation
48  * matrix, L is unit lower triangular, and U is upper triangular.
49  * The factored form of A is then used to solve the system of
50  * equations A * X = B.
51  */
