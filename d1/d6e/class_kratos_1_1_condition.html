virtual void Kratos::Condition::SetValueOnIntegrationPoints ( const Variable< double > &  rVariable,
std::vector< double > &  rValues,
const ProcessInfo &  rCurrentProcessInfo 
)
inlinevirtual

Access for variables on Integration points. This gives access to variables stored in the constitutive law on each integration point. Specialisations of condition must specify the actual interface to the integration points! Note, that these functions expect a std::vector of values for the specified variable type that contains a value for each integration point! SetValueOnIntegrationPoints: set the values for given Variable. GetValueOnIntegrationPoints: get the values for given Variable.
