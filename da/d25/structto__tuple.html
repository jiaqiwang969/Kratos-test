Detailed Description

template<typename ContainerType>
struct to_tuple< ContainerType >

A wrapper of a conversion function to convert a STL vector to a Python tuple. This class satisfies the requirements of the boost::python::to_python_converter conversion template argument.

Copied from scitbx/include/scitbx/boost_python/container_conversions.h that is described in the Boost.Python FAQ.
