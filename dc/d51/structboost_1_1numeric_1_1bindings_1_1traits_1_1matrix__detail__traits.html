Detailed Description

template<typename MIdentifier, typename MType>
struct boost::numeric::bindings::traits::matrix_detail_traits< MIdentifier, MType >

There is no default implementation since there is no reasonable default. Most matrix libraries provide totally different functions. Auxiliary traits class to reduce the number of specializations. MType is the actual matrix type. MIdentifier is used to specialize the traits for a specific matrix type, e.g. matrix_detail_traits< ublas::matrix<double>, ublas::matrix<double> const > matrix_detail_traits< ublas::matrix<double>, ublas::matrix<double> > Note that remove_const<MType>::type == MIdentifier
