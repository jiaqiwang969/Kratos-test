43  /*
44  * heev() computes the eigendecomposition of a N x N matrix
45  * A = Q * D * Q', where Q is a N x N unitary matrix and
46  * D is a diagonal matrix. The diagonal element D(i,i) is an
47  * eigenvalue of A and Q(:,i) is a corresponding eigenvector.
48  * The eigenvalues are stored in ascending order.
49  *
50  * On return of heev, A is overwritten by Q and w contains the main
51  * diagonal of D.
52  *
53  * int heev (char jobz, char uplo, A& a, W& w, minimal_workspace ) ;
54  * jobz : 'V' : compute eigenvectors
55  * 'N' : do not compute eigenvectors
56  * uplo : 'U' : only the upper triangular part of A is used on input.
57  * 'L' : only the lower triangular part of A is used on input.
58  */
