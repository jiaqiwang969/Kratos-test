Detailed Description

Short class definition.

Detail class definition.

Defines a linear elastic isotropic constitutive law in 3D space. This material law is defined by the parameters E (Young's modulus) and NU (Poisson ratio) As there are no further parameters the functionality is limited to linear elasticity.
