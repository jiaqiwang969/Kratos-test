44  /*
45  * hbev() computes the eigenvalues and optionally the associated
46  * eigenvectors of a banded Hermitian matrix A. A matrix is Hermitian
47  * when herm( A ) == A. When A is real, a Hermitian matrix is also
48  * called symmetric.
49  *
50  * The eigen decomposition is A = U S * herm(U) where U is a
51  * unitary matrix and S is a diagonal matrix. The eigenvalues of A
52  * are on the main diagonal of S. The eigenvalues are real.
53  *
54  * Workspace is organized following the arguments in the calling sequence.
55  * optimal_workspace() : for optimizing use of blas 3 kernels
56  * minimal_workspace() : minimum size of workarrays, but does not allow for optimization
57  * of blas 3 kernels
58  * workspace( work ) for real matrices where work is a real array with
59  * vector_size( work ) >= 3*matrix_size1( a ) - 2
60  * workspace( work, rwork ) for complex matrices where work is a complex
61  * array with vector_size( work ) >= matrix_size1( a )
62  * and rwork is a real array with
63  * vector_size( rwork ) >= 3 * matrix_size1( a ) - 2.
64  */
65 
66  /*
67  * If uplo=='L' only the lower triangular part is stored.
68  * If uplo=='U' only the upper triangular part is stored.
69  *
70  * The matrix is assumed to be stored in LAPACK band format, i.e.
71  * matrices are stored columnwise, in a compressed format so that when e.g. uplo=='U'
72  * the (i,j) element with j>=i is in position (i-j) + j * (KD+1) + KD where KD is the
73  * half bandwidth of the matrix. For a triadiagonal matrix, KD=1, for a diagonal matrix
74  * KD=0.
75  * When uplo=='L', the (i,j) element with j>=i is in position (i-j) + j * (KD+1).
76  *
77  * The matrix A is thus a rectangular matrix with KD+1 rows and N columns.
78  */
