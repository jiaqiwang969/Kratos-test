43  /*
44  * spsv() computes the solution to a system of linear equations
45  * A * X = B, where A is an N-by-N symmetric matrix stored in packed
46  * format and X and B are N-by-NRHS matrices.
47  *
48  * The diagonal pivoting method is used to factor A as
49  * A = U * D * U^T, if UPLO = 'U',
50  * A = L * D * L^T, if UPLO = 'L',
51  * where U (or L) is a product of permutation and unit upper
52  * (lower) triangular matrices, and D is symmetric and block
53  * diagonal with 1-by-1 and 2-by-2 diagonal blocks.
54  * The factored form of A is then used to solve the system
55  * of equations A * X = B.
56  */
