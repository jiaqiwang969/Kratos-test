Detailed Description

Defines a hyperelastic constitutive law in 3D space. This material law, also called Green-elastic material postulates the existance of a Helmholtz free-energy function (psi), which is defined per unit reference volume. For convenience, the strain-energy function vanishes in the reference configuration: psi(I)=0 "is defined by the parameters E (Young's modulus) and NU (Poisson ratio), and also, by the evaluation of the Helmholtz free-energy function. As there are no further parameters the functionality is applied to non-linear elasticity.
