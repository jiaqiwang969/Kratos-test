Detailed Description

template<class TContainerType>
class Kratos::Python::ContainerFromPython< TContainerType >

A sequence converter from python.

Basically this class is a modified copy from scitbx/include/scitbx/boost_python/container_conversions.h. It's changed to work with index_suit method instead of scitbx conversion policies.
