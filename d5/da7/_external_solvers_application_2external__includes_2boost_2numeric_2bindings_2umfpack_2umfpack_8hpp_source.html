327  /*
328  * Given a sparse matrix A in column-oriented form, and a symbolic analysis
329  * computed by umfpack_*_*symbolic, the umfpack_*_numeric routine performs
330  * the numerical factorisation, PAQ=LU, PRAQ=LU, or P(R\A)Q=LU, where P
331  * and Q are permutation matrices (represented as permutation vectors),
332  * R is the row scaling, L is unit-lower triangular, and U is upper
333  * triangular. This is required before the system Ax=b (or other related
334  * linear systems) can be solved.
335  */
