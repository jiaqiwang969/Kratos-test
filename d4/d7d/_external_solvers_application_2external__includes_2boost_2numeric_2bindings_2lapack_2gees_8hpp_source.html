44  /*
45  * gees() computes a Schur factorization of an N-by-N matrix A.
46  *
47  * The Schur decomposition is A = U S * herm(U) where U is a
48  * unitary matrix and S is upper triangular. The eigenvalues of A
49  * are on the main diagonal of S. If A is real, S is in pseudo
50  * upper triangular form.
51  *
52  * Workspace is organized following the arguments in the calling sequence.
53  * optimal_workspace() : for optimizing use of blas 3 kernels
54  * minimal_workspace() : minimum size of workarrays, but does not allow for optimization
55  * of blas 3 kernels
56  * workspace( work ) for real matrices where work is a real array with
57  * vector_size( work ) >= 3*matrix_size1( a )
58  * workspace( work, rwork ) for complex matrices where work is a complex
59  * array with vector_size( work ) >= 2*matrix_size1( a )
60  * and rwork is a real array with
61  * vector_size( rwork ) >= matrix_size1( a ).
62  */
