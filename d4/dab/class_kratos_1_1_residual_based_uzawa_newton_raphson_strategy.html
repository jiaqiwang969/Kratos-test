Detailed Description

template<class TSparseSpace, class TDenseSpace, class TLinearSolver>
class Kratos::ResidualBasedUzawaNewtonRaphsonStrategy< TSparseSpace, TDenseSpace, TLinearSolver >

Uzawa strategy for the solution of structural contact problems. This strategy implements an uzawa algorithm for the solution of contact problems using the augmented lagrange multiplier method.
