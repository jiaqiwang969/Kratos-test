Detailed Description

template<class TPointType>
class Kratos::Quadrilateral2D9< TPointType >

A nine node quadrilateral geometry. While the shape functions are only defined in 2D it is possible to define an arbitrary orientation in space. Thus it can be used for defining surfaces on 3D elements.
