183  /*
184  * hesv() computes the solution to a system of linear equations
185  * A * X = B, where A is an N-by-N Hermitian matrix and X and B
186  * are N-by-NRHS matrices.
187  *
188  * The diagonal pivoting method is used to factor A as
189  * A = U * D * U^H, if UPLO = 'U',
190  * A = L * D * L^H, if UPLO = 'L',
191  * where U (or L) is a product of permutation and unit upper
192  * (lower) triangular matrices, and D is Hermitian and block
193  * diagonal with 1-by-1 and 2-by-2 diagonal blocks. The factored
194  * form of A is then used to solve the system of equations A * X = B.
195  */
196
