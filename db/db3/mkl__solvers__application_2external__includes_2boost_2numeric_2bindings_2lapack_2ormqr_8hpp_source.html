39  // Apply the orthogonal transformation abtained by geqrf() to

44  /*
45  * ormqr() overwrites the general M by N matrix C with
46  *
47  * SIDE = 'L' SIDE = 'R'
48  * TRANS = 'N': Q * C C * Q
49  * TRANS = 'C' || 'T': Q**H * C C * Q**H
50  *
51  * where Q is a complex unitary matrix defined as the product of k
52  * elementary reflectors
53  *
54  * Q = H(1) H(2) . . . H(k)
55  *
56  * as returned by geqrf(). Q is of order M if SIDE = 'L' and of order N
57  * if SIDE = 'R'.
58  *
59  * Workspace is organized following the arguments in the calling sequence.
60  * optimal_workspace() : for optimizing use of blas 3 kernels
61  * minimal_workspace() : minimum size of workarrays, but does not allow for optimization
62  * of blas 3 kernels
63  * workspace( work ) where is an array that is used as auxiliary memory with the same value_type
64  * as C.
65  * We must have that vector_size( work ) >= matrix_size2( c )
66  * if SIDE=='L' otherwise vector_size( work ) >= matrix_size1( c )
67  */
