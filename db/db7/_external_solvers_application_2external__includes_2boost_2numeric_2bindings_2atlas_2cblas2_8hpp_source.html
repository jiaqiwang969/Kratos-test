235  // [from 'dspmv.f' (description assumes column major order):]
236  /* A is an array of DIMENSION ( ( n*( n + 1 ) )/2 ).
237  * Before entry with UPLO = 'U' or 'u', the array A must contain
238  * the upper triangular part of the symmetric matrix packed
239  * sequentially, column by column, so that A( 1 ) contains a(1,1),
240  * A( 2 ) and A( 3 ) contain a(1,2) and a(2,2) respectively, and
241  * so on.
242  * Before entry with UPLO = 'L' or 'l', the array A must contain
243  * the lower triangular part of the symmetric matrix packed
244  * sequentially, column by column, so that A( 1 ) contains a(1,1),
245  * A( 2 ) and A( 3 ) contain a(2,1) and a(3,1) respectively, and
246  * so on.
